1. Какой тип данных относится к строке в Си?
	а. string
	б. chars
	в. sym
	г. strlen
	д. ни один из вышеперечисленных вариантов

Ответ - д

2. Выберите корректные способы инициализации строк:
	а. string string = "Hello World!";
	б. char string[255] = "Hello World!";
	в. string string[] =  "Hello World!";
	г. char* string = "Hello World!";
	д. char string[] = {Hello World!};

Ответ - б, г

3. Выберите заголовочный файл предоставляющий стандартные функции для работы со строками:
	а. strio.h
	б. string.h
	в. strlib.h
	г. charlib.h
	д. basetyp.h

Ответ - б
